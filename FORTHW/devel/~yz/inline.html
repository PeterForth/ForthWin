inline.f  Вставка кода инлайном

Иногда требуется куда-то передать xt, но заводить для этого лишнее слово не хочется. Предлагаемые слова позволяют вставить код прямо в текущее слово.

(: ( компиляция: -- adr ref id; интерпретация: -- )

Начать компиляцию инлайна.

;) ( компиляция: adr ref id --; интерпретация: -- xt )

Закончить компиляцию инлайна и вернуть его xt. 


REQUIRE (:     ~yz/lib/inline.f
REQUIRE button ~yz/lib/winctl.f

  " Hello" button
