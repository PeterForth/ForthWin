api.f  Быстрое описание системных функций

Поскольку для любой типичной программы 90% внешних функций, вызываемых из динамических библиотек, приходится на три стандартные библиотеки Windows, имеет смысл упростить вызов, введя в дополнение к слову WINAPI: следующие слова:

KERNEL32: ( ->bl; -- )

USER32: ( ->bl; -- )

GDI32: ( ->bl; -- )
Пример

USER32: CreateMenu
KERNEL32: GlobalAlloc
