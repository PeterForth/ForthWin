Программы

Форт-система

• fres
Программа, преобразующая файлы формата .RES, созданные компилятором ресурсов, в файлы формата .FRES, которые можно подключать в двоичные форт-модули с помощью библиотеки RESOURCES.F

Пользовательский интерфейс

• xmenu  
Меню, не занимающее места на экране. Оно прячется к часам и, когда требуется, вызывается левой кнопкой мышки.
• db  
Универсальный пользовательский интерфейс к базам данным. Доступ через ODBC, тестировалось под MySQL.
• blank  
Попытка добавить к Windows простой командный язык, который позволял бы с минимумом затрат выводить на экран диалоговые окна с кнопочками, картинками и т.п. подобными признаками пользовательского интерфейса.

Верстка

• ged  
Пакетный графический редактор. Работает с картинками в формате .TIFF в моделях Grayscale и CMYK.
• tprint
Печать картинок в формате TIFF по несколько штук на листе. Для работы требуются внешние динамические библиотеки libtiff.dll, libjpeg.dll, zlib.dll.

Сети

• er22
Сервер протокола ER22. Протокол моего собственного изобретения смоделирован по образцу HTTP и предназначен для абстрагирования конкретных задач в виде единого интерфейса. Задачи выглядят как сетевые сервисы, отзывающиеся на вызовы с аргументами и возвращающие результаты. Сервисы могут обеспечиваться самим сервером или внешими задачами, запускаемыми на манер CGI.
