data.f  Работа с данными

init->> ( a -- )

Установить указатель

push->> ( -- a )

Запомнить текущий указатель в стеке

pop->> ( a -- )

Восстановить текущий указатель из стека

>> ( n -- )

Занести ячейку по адресу, на который указывает текущий указатель, и сдвинуть указатель

W>> ( w -- )

Занести двухбайтовое слово

С>> ( c -- )

Занести байт

2>> ( n1 n2 -- )

Занести две ячейки

N>> ( a n -- )

Занести n байтов, начинающихся с адреса а

Z>> ( z -- )

Занести asciiz-строку

z>> ( z -- )

Занести asciiz-строку, но не заканчивать ее нулем

zeroes>> ( n -- )

Обнулить следующиеn ячеек

DATA[ ( -- a )

Только для режима компиляции: начать вложенные данные

]DATA ( a -- )

Только для режима компиляции: закончить вложенные данные

Пример
: use-rec DATA[ 12 , 32 , 4 C, ]DATA 10 DUMP ;
