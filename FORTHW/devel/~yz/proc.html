proc.f  Процедуры

Исполняемые слова, по умолчанию не запускающиеся, а только возвращающие свой адрес. Удобны для объявления различного рода обработчиков, например для обработчиков событий Comevents.

PROC: name( ->bl; -- )

Вспомогательное слово: начинает компиляцию определения слова name. Полученное слово name при вызове вернет свой адрес xt.

PROC; ( -- )
